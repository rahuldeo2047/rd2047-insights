02 // hardware

Component Breakdown

Three major physical subsystems — each with updated 2025 alternatives where the community has found better or cheaper options.

  • 532nm Green DPSS Laser
    Used to pump the NV center from ground state, initializing the qubit to |0⟩. Requires power stability <0.5% to avoid initialization errors. Typical power: 50–300mW.
    Coherent OBIS 532nm or CNI MGL-III-532 — more affordable than legacy choices, USB-controllable
  • Acousto-Optic Modulator (AOM)
    Ultra-fast optical switch — pulses the laser in nanoseconds. Rise time matters: aim for <10ns. Acts as your quantum gate timing backbone.
    Gooch & Housego AOMO 3110-120 — common in open-source lab builds; Qubig also has compact options
  • Dichroic Mirror / Beam Splitter
    Reflects 532nm green, transmits 630–800nm red NV emission. Critical for separating excitation path from detection path without mixing signals.
  • Objective Lens (High NA)
    Focuses laser spot to sub-micron size on the diamond. Requires NA ≥ 0.9. Oil-immersion objectives (NA 1.3–1.4) maximize photon collection efficiency.
    Olympus UPLXAPO 100× / Nikon CFI Plan Apo — used in most academic DIY builds
  • Avalanche Photodiode (APD)
    Single-photon counting module (SPCM) — the "eye" that reads qubit state by counting individual red photons. Dark count rate <100 cps is ideal.
    Excelitas SPCM-AQRH — industry standard for DIY; PerkinElmer equivalent also well-documented
  • Optical Breadboard & Isolation
    Stainless M6-threaded table, 12–25mm thick. Must sit on sorbothane vibration pads (or optical table legs). Walking footsteps must not be detectable by APD.
// Optical Subsystem — Signal Flow // === EXCITATION PATH === Laser → AOM → Mirror → Objective ↓ Diamond (NV absorbs @ 532nm) // === EMISSION PATH === Diamond → Objective → Dichroic ↓ Longpass Filter (remove 532nm leakage) ↓ APD (counts photons @ 637–800nm) // === KEY PARAMS === ZPL = 637nm // Zero phonon line PSB = ~700nm // Phonon sideband peak NA_min = 0.9 // Minimum objective NA LaserΔP < 0.5% // Max power drift // === 2025 UPGRADE === // Consider fiber-coupled laser input // → eliminates pointing instability // → UncutGem project uses this approach
  • Microwave Signal Generator
    Must produce clean 2.87 GHz — the NV center zero-field splitting frequency. Frequency accuracy <1 kHz drift is important. Commercial generators are preferred over SDR for stability.
    Windfreak SynthHD Pro — $500, 54MHz–13.6GHz, USB-controlled; far cheaper than HP/Rohde & Schwarz at <1Hz resolution
  • RF Switch (SPDT)
    Chops continuous microwave tone into nanosecond pulses. Switching speed must be <5ns. Essential for defining Pi and Pi/2 pulse durations precisely.
    Mini-Circuits ZASWA-2-50DRA+ — 50Ω, <3ns switching, well-documented in community builds
  • RF Power Amplifier
    Boosts microwave signal to ≥20 dBm at antenna. Gain stability matters — thermal drift in amplifiers causes Rabi frequency changes over time. Include RF shielding.
    Mini-Circuits ZVE-3W-83+ — 2–8GHz, 33dBm output, affordable class for DIY; ZHL-16W-43-S+ for higher power
  • Copper Loop Antenna (Custom)
    25–50 micron wire loop, 1–3mm diameter, suspended 0.1–1mm above diamond surface. Delivers the oscillating magnetic field that flips the electron spin. Must be impedance-matched to 50Ω.
    UncutGem open-source PCB integrates antenna geometry into KiCad design — eliminates freehand wire loop
  • SMA Coaxial Cables & Connectors
    Phase-stable SMA cables rated to 18GHz+. Poor connectors cause signal reflections (VSWR) that distort pulse shapes and corrupt gate fidelity. Torque all SMA to spec (7 in-lb).

  • Pulse Sequencer / FPGA Board
    The real-time clock of your quantum computer. A standard OS (Windows/Linux) has millisecond jitter — far too slow for nanosecond gates. A dedicated FPGA runs deterministic timing with ~1ns resolution.
    SpinCore PulseBlasterESR-PRO (classic) OR Qudi-compatible Sinara/ARTIQ FPGA — full open-source ecosystem; community-standard for NV labs
  • Data Acquisition Card (DAQ)
    Reads TTL pulses from the APD photon counter and timestamps them with nanosecond resolution. Must support counter/timer mode, not just analog input. PCI or PCIe format preferred.
    NI PCIe-6321 or NI USB-6341 — widely used in open-source Qudi software; NI-DAQmx driver has Python bindings
  • Control PC
    Runs experiment orchestration software. Requires real PCIe slots (for DAQ/FPGA cards), USB 3.0, and reliable clock synchronization. Dedicated machine only — no background processes allowed during experiments.
    Any modern desktop with PCIe ×4 slots; pair with Qudi (open-source Python framework from Ulm University)
  • Digital Oscilloscope (>1GSa/s)
    For debugging pulse shapes, timing jitter, and microwave envelope fidelity. Minimum 500MHz bandwidth. A bad pulse has a bad gate — you need to see it before you trust it.
    Rigol DHO924S — 250MHz, 2GSa/s, ~$400; dramatically undercuts Tektronix/Keysight for DIY use
  • Control Software (Open Source)
    Qudi (Ulm University) is the community-standard Python framework for NV center control experiments. Provides GUI, hardware abstraction, and experiment modules. Alternative: pylabmaster, qcodes.
    GitHub: Ulm-IQO/qudi — Apache 2.0 license; active community; replaces all commercial LabVIEW-based alternatives

  • CVD Diamond (Type IIa)
    Chemical Vapor Deposition diamond with ultra-low nitrogen background (<5 ppb). Natural diamond has too many unwanted nitrogen atoms causing decoherence. Electronic grade Type IIa is the standard substrate.
    Element Six or Adamas Nanotechnologies — both supply electronic-grade CVD diamonds for research; 2×2×0.5mm chips, ~$200–500
  • NV Center Creation
    NV centers are artificially created by nitrogen ion implantation followed by vacuum annealing at >700°C. For a first build, purchase diamonds with pre-implanted NV centers from suppliers.
    Adamas Nano sells diamonds with pre-created, specified NV-center densities — skips need for ion implanter entirely
  • 13C-Enriched Diamond (Advanced)
    Standard diamond has 1.1% carbon-13 (nuclear spin) which causes decoherence. Isotopically purified diamonds with <0.01% 13C extend coherence times from microseconds to milliseconds.
    Diatope (Germany) supplies isotope-enriched CVD diamonds — coherence time improvement of 100× over natural carbon isotope ratio
  • SnV Center Alternative (Emerging)
    Tin-Vacancy (SnV) centers — a 2024 Fujitsu research highlight — offer 10× higher optical emission than NV centers. Less electric-field noise, better for photonic integration. Not yet DIY-accessible but worth watching.

03 // environment

Physical Environment

A standard desk, bedroom, or shared lab bench will not work. The environment must be engineered as carefully as the hardware.

🏗️ Vibration Isolation
  • Heavy breadboard minimum: 300mm × 450mm × 12.5mm aluminum, M6 threaded, resting on 4× sorbothane pads. Footstep vibration will shift a laser spot by micrometers — enough to lose the NV entirely.
  • Pneumatic table optional upgrade: Newport or TMC optical isolation tables cancel broadband vibration at 1–100Hz. Expensive but transformative for stability.
  • 3D-printed mounts viable: Stegemann et al. (2023, Eur. J. Phys.) demonstrated a full low-cost 3D-printed NV setup — total BOM under €5,000.
🌑 Optical Darkness
  • APD blind from ambient light: A power-strip LED or monitor glow will saturate your single-photon detector. You need complete blackout during measurements.
  • Custom blackout enclosure: Build a plywood or aluminum box around the breadboard lined with blackout fabric. Velcro cable entry ports. Add interlocked door sensor to cut laser power if opened.
  • No indicator LEDs: Remove or cover all LEDs on any instrument inside the enclosure. Power supply status lights are a common overlooked contamination source.
🌡️ Thermal Stability
  • Metal mounts drift with temperature: A 1°C change in a standard 50mm aluminum post causes ~1µm expansion — enough to defocus your laser spot.
  • No HVAC vents directly blowing on setup. Even slight air currents cause drift. Consider a closed room with passive thermal stabilization.
  • Invar or titanium mounts: For high-stability builds, use Invar-36 alloy mounts (CTE ≈ 1.2 ppm/°C vs 23 for aluminum). More expensive but worth it for 24/7 operation.
🔒 RF Shielding
  • Microwave leakage is real: Unshielded RF amplifiers at 3GHz will interfere with Wi-Fi, Bluetooth, medical devices, and neighboring electronics in a lab.
  • Copper mesh Faraday cage around the microwave chain keeps emissions contained and protects the NV's spin from external RF noise.
  • Mu-metal magnetic shielding: External DC magnetic fields shift the NV resonance frequency. Mu-metal cylinders around the diamond region reduce environmental B-field noise by 1000×.
04 // safety
⚠️
Critical Safety Hazards
This is not a software project. Physical risks are real and permanent. Do not skip this section.
🔴 Laser — Class 3B/4 Hazard

532nm green laser causes instant, irreversible retinal damage from direct or reflected beams. Back-reflections from optical surfaces are invisible until they hit your eye. OD5+ certified goggles specific to 532nm are mandatory at all times. Never look into an unshielded beam path. Use beam blocks and enclosed enclosure.

🟠 Microwave Radiation

Operating an unshielded antenna with an RF amplifier at 3GHz can cause localized tissue heating. At high power settings, this constitutes non-ionizing radiation exposure. Use minimum power needed for Rabi flipping, shield the amplifier output, and never operate with an open antenna next to body parts.

🟡 High Voltage Components

Avalanche Photodiodes (APDs) operate at reverse bias voltages of 200–400V. Some laser drivers have high-voltage stages. Never touch exposed high-voltage terminals. Ground all chassis. Use insulated test probes. Discharge capacitors before servicing.

🔵 Optical Component Cleaning

Contaminated objective lenses or diamond surfaces will destroy optical contrast. Do not touch optical surfaces bare-handed. Use first-contact polymer cleaner for objectives. Store the diamond in nitrogen or dry box when not in use. Surface contamination is not reversible for high-NA optics.

05 // roadmap

Recommended Build Sequence

Do not build all systems in parallel. Each phase validates the previous. Skipping steps creates impossible-to-debug compound failures.

01
Simulate first — use NVIDIA CUDA-Q Dynamics
Before spending money on hardware, model your pulse sequences using CUDA-Q Dynamics (free, Python). Validate your understanding of Rabi oscillations, T2 decay, and ODMR spectra in simulation. This reveals design errors before they become expensive hardware problems.
Week 1–2
02
Study UncutGem open-source reference design
The QuantumVillage UncutGem project (github.com/QuantumVillage/UncutGem) provides fully open KiCad PCB schematics, firmware, and a bill of materials for a working NV center magnetometer. Use this as your hardware reference before designing anything custom.
Week 2–3
03
Build optical path first — verify ODMR before microwaves
Set up the confocal microscope, verify you can see NV fluorescence under green laser illumination, and collect a photoluminescence map of the diamond surface. Confirm photon counts are stable. Only then proceed to add microwave hardware.
Month 1–2
04
Measure ODMR spectrum — confirm qubit resonance
Apply a weak, swept microwave tone while monitoring APD counts. You should observe a dip in fluorescence at 2.87 GHz — this is the electron spin resonance (ESR) signature confirming your NV center is functioning as a qubit. Apply an external DC magnetic field to split the two dips (ms = ±1).
Month 2–3
05
Demonstrate Rabi oscillations — first quantum gate
Apply pulsed microwave bursts of increasing duration and measure photon counts after each. A sinusoidal oscillation (Rabi flop) confirms coherent spin control. The first minimum is your π-pulse — your first quantum NOT gate. This is the foundational milestone.
Month 3–4
06
Measure T2 coherence — optimize environment
Run a Hahn echo experiment to measure T2 spin coherence time. This directly measures how long your qubit retains quantum information. Improvements here come from: better vibration isolation, lower background nitrogen in diamond, isotopically pure carbon, and reduced magnetic noise.
Month 4–6
