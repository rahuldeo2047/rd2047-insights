04 // setup

Complete Setup Checklist

Everything you need to go from placeholder — to real live numbers.

Step-by-step: activate your analytics
All free. No paid tools required.
1
Go to analytics.google.com → Admin → Create Property. Enter your site name and timezone. Click Next → Create.
2
Under the new property → Data Streams → Web → enter your URL → Create Stream. You'll get a Measurement ID like G-8Y8EYHB278.
3
In every HTML file in this project, find G-8Y8EYHB278 and replace it with your real ID. Files: index.html, nv-quantum-requirements.html, safe-qc-system.html, sarvam-ai-edge-computing.html, consciousness-and-quantum-collapse-correlation-testing.html, stats.html.
4
Deploy your files to a web host (GitHub Pages, Netlify, Vercel — all free). Visit your pages. Check GA4 Reports → Realtime to verify tracking is working.
5
(For live embed here) Go to lookerstudio.google.com → Create → Report → connect your GA4 property. Build a pages table or bar chart. Click Share → Embed report → copy the <iframe> tag and paste it into the embed section above in this file.
6
Add a link to this stats.html page in your site footer so visitors can see public stats. GA4 data is private to you — only what you choose to embed here is public.
